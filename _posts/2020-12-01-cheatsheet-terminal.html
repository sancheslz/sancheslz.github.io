Navegação
pwd
Exibe o caminho do diretório atual
cd
Muda para o diretório de início do Terminal
cd ..
Volta o diretório em um nível
cd ~
Retorna ao diretório-base do usuário
cd /
Retorna ao diretório raiz
cd -
Retorna ao último diretório acessado
cd /caminho/do/diretorio
Muda para o diretório especificado
Arquivos e Diretórios
mkdir nome_pasta
Criar uma pasta
rmdir nome_pasta
Remove uma pasta (vazia)
rm arquivo
Remove um arquivo
mv origem destino
Move/Renomeia o arquivo
cp origem destino
Copia um arquivo
<comando> -i
Confirma antes de executar (evita sobrescrição)
<comando> -r
Opera de modo recursivo
<comando> -v
Exibe a lista das operações realizadas
Listagem de Arquivos
ls
Lista os itens do diretório
ls -a
Inclúi na exibição os arquivos ocultos
ls -l
Exibe uma lista com detalhes dos arquivos
ls -F
Exibe o tipo do arquivo (inclui / nos diretórios)
ls -G
Exibe com cores para cada tipo de arquivo
ls -lh
Exibe o tamanho do arquivo em formato legível ao usuário
ls -r
Exibe os itens em order reversa
ls -S
Exibe os itens por ordem de tamanho
ls -S
Exibe os itens por ordem da última modificação
ls pasta1 pasta2
Exibe os itens das duas pastas
ls -i
Exibe o hash do arquivo, isto é, seu valor na memória do computador
Caracteres Curinga
*
Quaisquer caracteres
?
Quaisquer um caracter
[char]
Quaisquer caracter do conjunto
[!char]
Quaisquer caracter fora do conjunto
[[:class:]]
Quaisquer caracter de uma classe
Classes de Caracteres
[:alnum:]
Qualquer caracter alfanumérico
[:alpha:]
Qualquer caracter alfabético
[:digit:]
Qualquer número
[:lower:]
Qualquer caracter lowercase
[:upper:]
Qualquer caracter uppercase
[:word:]
Semelhante ao [:alnum:] mas adiciona o _
[:blank:]
Inclui espaços e tabulações
[:cntrl:]
Caracteres de controle (ASCII de 33 a 126)
[:graph:]
Os caracteres visíveis (ASCII de 33 a 126)
[:punct:]
Caracteres de pontuação
[:print:]
Caracteres imprimíveis (todos de [:graph:] mais o espaço )
[:space:]
Todos os caracteres que representam um espaço (\t \r \n \v \f)
[:xdigit:]
Caracteres que representam os hexadecimais
Obter Ajuda
man <comando>
Abre o manual de ajuda sobre o comando
<comando> --help
Exibe informações rápidas de ajuda (se disponível)
apropos <comando>
Retorna a seção do manual em que se encontra o comando
whatis <comando>
Exibe uma breve descriçao do comando e sua página no manual
info <comando>
Retorna as informações principais sobre determinado comando
Criando Alias
alias <apelido>="<comando1;comando2>"
Cria um alias na seção do Terminal
type <apelido>
Verifica se o alias está disponível para ser criado
unalias <apelido>
Exclui o alias criado
alias
Exibe a lista de alias cadastrados
Lendo Arquivos
cat <arquivo>
Exibe o conteúdo do arquiv
less <arquivo>
Exibe o conteúdo do arquivo
sort <arquivo>
Exibe o conteúdo em ordem alfabética
uniq <arquivo>
Exibe apenas as linhas únicas do arquivo
tail <arquivo>
Exibe apenas o final do arquivo
head <arquivo>
Exibe apenas o começo do arquivo
tee <arquivo>
Permite a inserção no arquivo de modo interativo, retornando a saída padrão
Usando o Echo
echo <argumento>
Retorna o resultado dos argumentos passados
echo $((2+2))
Retorna o resultado da equação passada
echo texto-{A,B,C}
Retorna múltiplos textos com o padrão especificado
echo texto-{1..5}
Retorna múltiplas textos com o intervalo especificado. Pode ser letra ou números
echo text{A{1,2},B{3,4}}
Retorna o padrão das chaves aninhadas
echo $USER
Retorna o valor da variável, neste exemplo, o usuário
echo $(cal)
Retorna o restuldado do parâmetro passado
